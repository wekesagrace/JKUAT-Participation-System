# SautiYetu – AI-Powered Public Participation Analysis System
Project Overview

SautiYetu is an AI-powered public participation analysis system designed to improve governance transparency and stakeholder engagement.

The system collects feedback from students and staff through multiple channels including web forms, SMS, and voice transcription, then automatically analyses the feedback using Natural Language Processing (NLP).

Objectives

- Collect feedback through multiple channels.
- Perform automated sentiment analysis.
- Extract recurring discussion topics.
- Provide real-time dashboards for administrators.
- Support evidence-based decision making.

Features

- Web-based feedback submission
- SMS feedback integration
- Audio-to-text transcription
- Sentiment Analysis
- Topic Modelling
- Admin Dashboard
- Event Management
- CSV Export
- Email Alerts
- Anonymous Feedback Submission

Installation

1. Clone repository
git clone https://github.com/yourusername/sautiyetu.git

2. Install dependencies
pip install flask flask-sqlalchemy textblob scikit-learn whisper africastalking flask-mail openpyxl reportlab

3. Download TextBlob corpora
python -m textblob.download_corpora

4. Run application
python app.py
